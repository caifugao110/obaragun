# obaragun
焊枪在线选型数据库测试版本

## 项目简介

小原焊枪选型数据库是一个专为客户提供焊枪选型服务的在线平台，旨在帮助用户快速、准确地找到符合其需求的焊枪产品。该系统提供了详细的焊枪参数查询、图片预览和模型下载功能，方便用户进行产品选型和评估。

## 功能特点

- **用户登录系统**：提供安全的用户登录界面
- **多分类焊枪查询**：支持NOC、NOX、UC、UX等多种焊枪分类
- **详细参数筛选**：可根据喉深、喉宽、加压力等参数进行精确筛选
- **产品图片预览**：支持点击查看焊枪大图
- **模型下载功能**：提供CAD和STEP格式的模型下载
- **响应式设计**：适配不同设备屏幕尺寸

## 技术栈

- **前端框架**：Bootstrap 3.3.7
- **JavaScript库**：jQuery
- **CSS框架**：Pure CSS
- **图片查看**：jQuery.imgbox
- **后端框架**：Express.js
- **数据库**：内存存储（模拟数据库）
- **认证**：Session认证
- **密码加密**：bcrypt

## 安装与运行

1. **克隆项目**：
   ```bash
   git clone <项目仓库地址>
   cd obaragun
   ```

2. **安装依赖**：
   ```bash
   npm install
   ```

3. **启动服务器**：
   ```bash
   npm start
   ```

4. **访问系统**：打开浏览器，输入 `http://localhost:3000`，进入登录页面

## 使用方法

1. **用户登录**：输入用户名和密码，点击登录按钮
   - 默认管理员账号：admin / admin123
   - 默认用户账号：user / user123

2. **选择焊枪分类**：在导航栏中选择需要查询的焊枪分类（如NOC、NOX等）

3. **设置筛选条件**：根据需要设置喉深、喉宽、加压力等参数范围

4. **提交查询**：点击"我已选好，确认提交"按钮，查看搜索结果

5. **查看详情**：点击图片预览查看焊枪大图

6. **下载模型**：点击CAD下载或STEP下载按钮获取模型文件

7. **查看搜索历史**：系统会记录用户的搜索历史，方便用户查看之前的查询结果

8. **收藏焊枪**：用户可以收藏常用的焊枪，方便后续查看
